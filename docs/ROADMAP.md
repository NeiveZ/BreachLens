# Roadmap

## Version 1.x

- CSV export
- SQLite report history
- richer scoring configuration
- provider plugin interface
- Slack/Discord/Telegram notifications

## Version 2.x

- FastAPI dashboard
- scheduled monitoring mode
- trend charts by domain and severity
- authenticated team workspace
- additional authorized data providers

## Non-goals

- dark-web market scraping
- leaked database ingestion
- credential stuffing automation
- plaintext credential collection
